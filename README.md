# Finals Checklist · برنامه مطالعه فاینال

A personal study tracker for final exams — built for Tir ترم چهار.

## Features

- ✅ Check off lessons as you study
- ➕ Add custom lessons to any subject
- ✕ Remove lessons you don't need
- 📊 Progress bar tracks overall completion
- 💾 Progress auto-saves in the browser (localStorage)

## Subjects

| Subject | Exam Date |
|---|---|
| 听力与阅读 — Listening & Reading | Tir 20 |
| 综合 — Comprehensive Chinese | Tir 22 |
| 写作 — Writing | Tir 24 |
| 口语 — Speaking | Tir 29 |

## Deploy to GitHub Pages

1. Create a new repo on GitHub (e.g. `finals-checklist`)
2. Upload the files in this folder
3. Go to **Settings → Pages → Source → main branch / root**
4. Your checklist will be live at `https://yourusername.github.io/finals-checklist`

## Local use

Just open `index.html` in any browser — no server needed.
